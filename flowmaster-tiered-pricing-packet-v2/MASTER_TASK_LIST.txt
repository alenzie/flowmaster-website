TASK 05: TIER SELECTOR UI (ARCADE CAROUSEL)
─────────────────────────────────────────
DESIGN DECISIONS NEEDED:

  1. Animation Timing & Easing
     - Float animation duration (spec says 3s, confirm)
     - Shine effect speed (spec says 3s, may feel slow)
     - Card transition timing when selecting

  2. Mobile Behavior
     - Touch/swipe gesture support?
     - Card sizing on small screens
     - Navigation arrows vs swipe-only on mobile

  3. Color Tuning
     - Exact glow intensity for active tier
     - Gray-out level for sold-out tiers
     - Particle color/density balance

  4. Edge Cases
     - What if ALL tiers are sold out?
     - What if user has slow connection during checkout redirect?
     - What happens on checkout cancellation redirect?

  5. Accessibility
     - Keyboard navigation (arrow keys) - specified but verify
     - Screen reader announcements for tier changes
     - Focus management after modal/checkout

RECOMMENDED APPROACH:
  → Build static version first (no animations)
  → Add animations incrementally
  → Test on actual hardware before polish pass
  → Consider creating a Storybook/preview page for iteration

     - Test in Gmail, Outlook, Apple Mail
     - Mobile rendering verification
     - Dark mode compatibility
